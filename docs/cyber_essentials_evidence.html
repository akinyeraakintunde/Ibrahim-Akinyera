Cyber Essentials Certification – Connected Health

Evidence document – Tech Nation Global Talent (Digital Technology)
Prepared by: Ibrahim Akinyera

1. Context and Objective

Connected Health is a healthcare organisation operating in a regulated environment with sensitive data and strict requirements around security and compliance. During my time with the organisation, I contributed to the technical preparation and implementation of controls required for the UK Government–backed Cyber Essentials certification.

2. Areas of Contribution

2.1 Secure Configuration and System Hardening

  • Supporting OS and application patch validation for key systems and devices.
  • Helping to remove outdated or non-compliant software from endpoints.
  • Assisting in implementing secure baselines on Windows endpoints.

2.2 Access Control and Identity Management

  • Reviewing user account lists and helping identify unused or orphaned accounts.
  • Supporting the validation of privileged account usage.
  • Ensuring multi-factor authentication and strong authentication practices where applicable.

2.3 Firewall, Network and Endpoint Security

  • Supporting checks on firewall configurations and endpoint firewall rules.
  • Contributing to the validation of permitted inbound and outbound network traffic.
  • Confirming that endpoints had up-to-date antivirus and malware protection.

2.4 Documentation and Audit Evidence

  • Helping prepare inventories of systems and devices.
  • Supporting the collation of screenshots and configuration evidence.
  • Contributing to internal responses for the Cyber Essentials questionnaire.

3. Impact

These activities helped Connected Health:

  • Reduce exposure to avoidable security risks.
  • Strengthen its cyber posture across endpoints and accounts.
  • Provide credible technical evidence to the certification assessors.
  • Successfully achieve Cyber Essentials certification.

4. Relevance to Tech Nation

This experience demonstrates my ability to:

  • Work within a regulated, security-sensitive environment.
  • Contribute meaningfully to cyber governance and certification activities.
  • Engage with practical secure configuration and endpoint hardening work.
