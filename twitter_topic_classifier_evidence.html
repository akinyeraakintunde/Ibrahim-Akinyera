Twitter Topic Classifier (Machine Learning Project)

Evidence document – Tech Nation Global Talent (Digital Technology)
Prepared by: Ibrahim Akinyera

1. Project Overview

The Twitter Topic Classifier is a machine learning project that classifies tweets into thematic categories such as politics, sport, entertainment and other domains. It demonstrates my ability to design and implement an end-to-end ML pipeline, from data preparation and feature extraction through to training, evaluation and deployment in a reproducible GitHub repository.

2. Problem Statement

Social media platforms like Twitter contain rich, real-time signals about public discourse. Understanding the thematic focus of tweets at scale is valuable for media monitoring, trend analysis and business intelligence. However, tweets are short, noisy and unstructured.

The objective of this project is to build a classifier that can automatically label tweets by theme, using a robust ML pipeline and best practices in applied natural language processing.

3. Technical Approach

3.1 Data Preparation

  • Collection of labelled tweet data from academic or open datasets.
  • Cleaning and normalisation (lowercasing, URL removal, basic token cleaning).
  • Handling of class balance where necessary.

3.2 Feature Engineering

The project uses classical text representations such as TF–IDF and bag-of-words models, which remain strong baselines for short-form text classification. This includes:

  • Tokenisation and stop-word handling.
  • Term frequency and inverse document frequency features.
  • Sparse vector representations for input into ML models.

3.3 Model Training and Evaluation

Several classifiers were evaluated, including logistic regression and tree-based models. The work focused on:

  • Train/test splits and cross-validation for reliable evaluation.
  • Metrics such as accuracy, F1 score and confusion matrices.
  • Identifying performance differences across themes (e.g. politics vs sport).

4. Repository and Reproducibility

The full project is hosted publicly: github.com/akinyeraakintunde/Twitter-Topic-Classifier-with-ML.

The repository contains:

  • Source code for data loading, feature extraction and model training.
  • Configuration and requirements for reproducible experiments.
  • A clear README explaining how to run and extend the project.
