Evidence 1: NxtAbroad AI – Intelligent Eligibility & Lead Scoring Engine

Demonstrates technical innovation and product-level engineering in AI-enabled EdTech.

Project Summary

NxtAbroad AI Core is a Python-based rules engine and lead-scoring system that automates student eligibility checks, visa risk classification and prioritisation for education and immigration advisory. It replaces manual spreadsheet-driven assessments with a consistent, auditable decision engine.

What the System Does

  • Ingests student lead data from CSV / CRM / web forms.
  • Applies explicit eligibility rules (CGPA, funds, IELTS, visa history).
  • Classifies risk (Low / Medium / High) based on rule violations.
  • Generates a 0–100 lead score using weighted academic, financial and engagement metrics.
  • Produces human-readable explanations for each decision.

Technical Architecture

  • config.py – country-level thresholds and scoring weights.
  • rules_engine.py – deterministic rules and risk classification.
  • lead_scoring.py – composite scoring model with risk penalty.
  • demo_run.py – CLI script to run the full pipeline against sample data.
  • docs/architecture.md – detailed technical documentation.

Impact & Relevance

This project shows my ability to design and implement a production-style decision engine, combining data modelling, rules design, scoring and documentation. It underpins NxtAbroad’s move towards a fully AI-powered recruitment platform and directly supports Tech Nation’s criteria for technical innovation and product-led impact.

Repository: github.com/akinyeraakintunde/nxtabroad-ai-core

Evidence 3: NLP Pipeline for Multi-Class Social Media Text Classification (MSc Project)

Demonstrates advanced technical depth in machine learning and natural language processing.

Project Summary

As part of my MSc Data Science at Ulster University, I built a full end-to-end NLP pipeline capable of classifying social media text into Politics, Sports and Entertainment categories. The project involved dataset creation, preprocessing, TF-IDF vectorisation, model training and evaluation using Python.

Technical Highlights

  • Custom dataset (~600 documents across 3 classes).
  • TF-IDF vectorisation with NLTK preprocessing.
  • Models: SVM, Logistic Regression, Naive Bayes.
  • Evaluation using accuracy, F1-score and confusion matrix.
  • Python implementation in script_model.py.
